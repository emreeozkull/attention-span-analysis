Analyzing Personal Attention Span Through Netflix Viewing Patterns

This project explores how personal attention span has changed over time by analyzing Netflix viewing history. The analysis provides insights into engagement levels, content preferences, and factors that influence shifts in attention.

Data Analysis Techniques

Time-Based Analysis

Examining viewing patterns over different periods (daily, weekly, monthly)

Session Analysis

Defining viewing sessions and analyzing duration/frequency

Content Analysis

Categorizing watched content by genre and type

Attention Metrics

Quantifying attention span through various metrics
